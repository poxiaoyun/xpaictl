## 重装环境
### 1. 登录服务服务器，执行下面命令

```
# 设置环境变量
# 注意defaultDir 是 xpai.yaml 中的平台安装目录

export defaultDir=/data
export SEALOS_RUNTIME_ROOT=${defaultDir}/.sealos
export SEALOS_SCP_CHECKSUM=false
export SEALOS_DATA_ROOT=${defaultDir}/registry

# 重置集群，选择 yes
sealos reset
```

#### 2. 清理脏数据

```
# 停止服务
systemctl stop registry image-cri-shim

# 删除/data/下安装目录脏数据
rm -rf /data/{openebs,registry,containerd,sealos}
rm -rf /var/lib/registry

# 删除和清空镜像，直到下面这样
sealos images
REPOSITORY                                               TAG        IMAGE ID       CREATED          SIZE
```

#### 3. 重新安装

```
./xpaictl.sh --config xpai.yaml --masters <服务器ip> --offline
```
