# 文档样式使用说明

## 概述

本文档说明如何使用 `styles.css` 中定义的样式来美化 Markdown 文档。

## 表格样式

### 自动应用

所有 HTML 表格会自动应用以下样式：

- ✅ **自动填满宽度**：表格宽度为 100%
- 🎨 **美观的表头**：蓝色渐变背景，白色文字
- ✨ **悬停效果**：鼠标悬停时行背景色变化
- 📱 **响应式设计**：小屏幕时自动横向滚动
- 🎯 **阴影效果**：轻微阴影和圆角

### 使用示例

#### 标准表格

```html
<table>
<thead>
<tr>
<th>列1</th>
<th>列2</th>
</tr>
</thead>
<tbody>
<tr>
<td>数据1</td>
<td>数据2</td>
</tr>
</tbody>
</table>
```

#### 斑马纹表格（可选）

添加 `striped` 类：

```html
<table class="striped">
<!-- 表格内容 -->
</table>
```

#### 紧凑型表格（可选）

添加 `compact` 类：

```html
<table class="compact">
<!-- 表格内容 -->
</table>
```

## 引用块样式

引用块已自动应用样式，支持以下类型：

- 💡 **提示**：绿色边框
- ℹ️ **说明**：蓝色边框
- ⚠️ **警告**：黄色边框
- 🚨 **危险**：红色边框
- ✅ **成功**：绿色边框

## 在 Markdown 渲染器中应用样式

### VS Code

1. 安装 Markdown Preview Enhanced 插件
2. 在设置中配置 CSS 文件路径

### GitHub/GitLab

GitHub 和 GitLab 会自动应用内联样式，HTML 表格会正常显示。

### 其他渲染器

大多数支持 HTML 的 Markdown 渲染器都会自动应用内联样式。如需应用外部 CSS 文件，请参考相应渲染器的文档。

## 注意事项

- 表格必须使用 HTML 格式（`<table>`），Markdown 表格语法不会应用这些样式
- 样式文件位于 `docs/styles.css`
- 所有样式都包含响应式设计，适配移动设备
